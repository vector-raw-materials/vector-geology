vector-geology¶

vector-geology is a Python package for Bayesian inversion of geophysical data.

Requirements¶

The requirements for the core functionality of the package are:

.
numpy
matplotlib
python-dotenv

Optional requirements¶

There are many optional requirements, depending on the data format you want to read/write. Currently, the requierements_opt.txt reads like:

git+https://github.com/terranigma-solutions/subsurface.git#egg=subsurface
omfvista
pyvista
